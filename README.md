# Travel Planner API

 В нашій базі дних є 2 тіблиці: Project і Place з якими ми працюємо

## Функціонал
- Створення/редагування/видалення туристичних проєктів.
- Додавання місць до проєктів з перевіркою через зовнішнє API.
- Валідація лімітів (макс. 10 місць на проєкт).
- Автоматичне відстеження статусу завершення проєкту.
- Заборона на видалення проєктів з відвіданими місцями.
